Cелекторы, псевдоклассы, псевдоэлементы

Селекторы - общее

  • * → универсальный селектор (для всех элементов)
  • .myclass p → все вложенные p в класс .myclass (любого уровня)
  • .myclass > p → только для вложенных в класс .myclas p 1ого порядка (прямые потомки)
  • p + a → для 1ого соседнего "a" идущего СРАЗУ после "p" на том же уровне (1ый сосед). Аналогично для классов, пример: .myclass + .twoclass
  • p ~ a → для соседних "a" идущих после "p" на том же уровне (соседи). Аналогично для классов, пример: .myclass ~ .twoclass
  • p, .myclass, h1 → группирование стилей
  • .myclass.twoclass → одновремменно 2 скласса (указываем без пробела). Пример: div class="myclass twoclass" - блок, содержащий оба класса
  • #myid.myclass → элемент с id=myid и c классом=myclass (указываем без пробела). Пример: div id="myid" class="myclass"

Селекторы - по атрибутам

  • div[attr] → div, у которого есть атрибут
  • div[attr="name"] → div, у которого атрибут = "name" (или любое другое значение атрибута, name - как пример)
  • div[attr^="val"] → div, у которого атрибут начинается с "val", например: val-new
  • div[attr|="val"] → div, у которого атрибут равен ИЛИ начинается с "val"
  • div[attr*="val"] → div, у которого атрибут содержин подстроку "val", например: myval, newvalTwo
  • div[attr~="val"] → div, у которого атрибут содержин "val" через пробел, например: attr="del val"
  • div[attr$="val"] → div, у которого атрибут заканчивается на "val", например: newval
  • div[attr$="val"] → div, у которого атрибут заканчивается на "val", например: newval

Псевдо-классы - общие

  • Псевдоклассы – это селекторы, которые определяют состояние уже существующих элементов, которое может меняться при определенных условиях.
  • :first-child → первый дочерний элемент
  • :last-child → последний дочерний элемент
  • :only-child → единственный дочерний элемент (когда соседних элементов нет)
  • :nth-child(5) → пятый дочерний элемент
  • :nth-child(2n) → четные дочерние элементы (2,4,6, ...)
  • :nth-child(n+2) → дочерние элементы, начиная со второго
  • :nth-child(2n+1) → нечетные дочерние элементы (1,3,4, ..)
  • :first-of-type → первый дочерний элемент заданного типа (в рамках прямого родителя)
  • :last-of-type → последний дочерний элемент заданного типа (в рамках прямого родителя)
  • :nth-of-type(3) → третий СОСЕД (3ий соседний элемент)
  • :nth-last-of-type(4) → 4ий сосед с конца
  • :empty для элементов, внутри которых нет ни одного символа (без текста)
  • :not(selector) → элементы, которые НЕ содержат заданный селектор (все, КРОМЕ заданного типа). Например: div:not(.myClass) - не содержат класс .myClass
  • ::lang(ru) → стили в зависимости от языка. например, для задания разных стилей кавычек в цитатах

Псевдо-классы - для ссылок

  • :link → непосещенные ссылки
  • :visited → посещенные ссылки
  • :active, :hover, :focus → активные (при нажатии), при наведении и при фокусе
  • :link:hover → наведение на непосещенную ссылку
  • :visited:hover → наведение на посещенную ссылку

Псевдо-классы - для элементов формы

  • :valid → содержание прошло проверку в браузере на соответствие (то есть, оно валидно)
  • :invalid → содержание не прошло проверку (не валидно)
  • ::required → элементы, у которых задан атрибут required (который указывает, что пользователь обязательно должен выбрать/добавить значение перед отправкой формы)
  • :disabled → заблокированное поле (не доступный элемент формы)
  • input:enabled → НЕ заблокированный элемент формы
  • :in-range → поля, значение которых входит в заданном диапазоне
  • :out-of-range → поля, значение которых НЕ входит в диапазон
  • input:checked → для переключателей в форме: отмеченные (включенные) флажки, радио инпуты..
  • input[type=radio]:checked → элемент с типом "radio", который был отмечен

Псевдо-элементы

  • Псевдоэлементы – это селекторы, которые определяют область элементов, которая изначально отсутствует в дереве документа. Эта область создается искусственно с помощью CSS.
  • ::before → добавить содержимое в начало
  • ::after → добавить содержимое в конец
  • С помощью before и after, можно сделать:
    маркеры у списков; символы для "хлебных крошек"; особые ковычки в тексте; подложка у блока (бэграунд, изображение, какие-то картинки смещенные по странице..); наведение на какой-то блок (появление текста, картинки или наложение цвета)
  • ::first-letter → стили для первой буквы
  • ::first-line → стили для первой строки
  • :hover::before → псевдоэллемент при наведении
  • ::selection → стили к контенту, выделенному мышкой
